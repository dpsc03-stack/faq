# よくある質問（FAQ）

公式LINEに寄せられる
よくある質問をまとめています。

---

## ① 事前準備について

**Q. どんなウォレットが必要ですか？**  
A. Trust Wallet または MetaMaskで問題ありません。

**Q. 用意する通貨は何ですか？**  
A. Baseチェーン上のETH（ガス代）です。

---

## ② 接続・設定について

**Q. スマホだけで設定できますか？**  
A. 基本的な設定は可能ですが、Zoomを使用する場合はPCを推奨しています。

---

## ③ トラブル・エラーについて

**Q. 送金が反映されません**  
A. ネットワーク混雑やガス代不足が原因の可能性があります。
